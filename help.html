Where's the chessboard?!

Komodo just calculates moves and relies on other programs to display the chessboard.

Steps to use Komodo with a chessboard:

  • Install the chessboard Arena Chess Gui
  • Add Komodo to Arena: Engines > Install New Engine watch video of this process.
